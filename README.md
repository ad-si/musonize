# Musonize

Musonize is an open source optical music recognition (OMR) preprocessor written in C++.

## Possible Steps

- Steps
- Convert to grayscale
- Otsus method
- Rotate
- Crop
- Remove noise
- Remove holes
- Scale to uniform size
- (Otsus method)
- Split into sections
- Remove lines
- Recognize each blob
- Reassociate with original position
- Generate MusicXML
